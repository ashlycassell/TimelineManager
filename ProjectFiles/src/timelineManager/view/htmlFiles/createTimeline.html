Creating a Timeline

  • Click on the add timeline button
  • Fill in the Title field of the new pop-up window
  • (Optional) Add a description
  • Select a Start and End date
  • Press Save to finish or Cancel/ESC/X button of the pop-up window to return

A timeline has been created and is currently displayed in the timeline viewer in a unique color. The timeline can be accessed at any time by directly clicking on it or its name (displayed in the Timelines list). Double clicking on the timeline name in the Timelines list will set the view focus onto that timeline. When hovering over the timeline with the mouse pointer a tooltip will display the details of the timeline.
