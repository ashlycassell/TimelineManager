Edit Timeline/Event

  • Right-click on the timeline/event
  • Select Edit Timeline/Task
  • Edit the fields of the pop-up window
  • Press Save to finish or Cancel/ESC/X button of the pop-up window to return

The timeline/event has been updated and will be displayed according to the new information.

Editing the start and/or end date of a timeline may affect the tasks of the timeline so make sure to edit the tasks beforehand!
